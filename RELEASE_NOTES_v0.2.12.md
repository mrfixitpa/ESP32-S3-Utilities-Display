# Release Notes – v0.2.12

## Summary
This release focuses on bringing the Water utility card fully online with real data and improving overall usefulness of the display without changing layout or design.

---

## What's New

### 💧 Water Integration (Complete)
- Replaced placeholder values with real Home Assistant sensors
- Added **weekly water usage**
- Updated Water card to display:
  - **Now** – live flow rate (GPM)
  - **Week** – total gallons used this week
  - **Month** – total gallons used this month

---

## Improvements
- Water data is now fully functional and actionable
- Maintains clean, consistent layout with existing cards
- Preserves original spacing and design intent

---

## No Changes
- Electricity card (unchanged, still live data)
- Gas card (still placeholder)
- Fonts, icons, and layout
- Performance and refresh behavior

---

## Notes
This release continues the transition from a visual prototype to a fully data-driven utility display.

Gas integration is planned next.
